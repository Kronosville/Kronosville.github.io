--- layout: kronosville title: Resource Pack base_loc: Home --- The resource pack is necessary for the Kronosville Christmas/New Year party.

Installation

  1. Download the resource pack.
  2. In-game, go to the Esc menu > Options... > Resource Packs... > Open resource pack folder.
  3. Drop the downloaded ZIP file in the opened folder.
  4. Press Esc while on the resource pack screen.
  5. Re-enter the resource pack screen and select the "kronosville.zip" resource pack.
  6. Hit Done.
